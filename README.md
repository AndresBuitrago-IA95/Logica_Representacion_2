# Lógica y Representación II - Universidad de Antioquia

Plataforma web interactiva para el curso de Lógica y Representación II de la Facultad de Ingeniería, Universidad de Antioquia (2026-1).

## 🎯 Características

- **Colores oficiales UdeA**: Verde #00A859 y blanco
- **7 unidades temáticas completas** con teoría detallada
- **Laboratorio Python integrado** con Pyodide (Python en el navegador)
- **Visualizaciones interactivas** para cada estructura de datos
- **Ejemplos de código ejecutables** en tiempo real
- **Explicaciones paso a paso** de algoritmos y métodos

## 📚 Contenido del Curso

### Semanas 1-3: Listas Ligadas
- Listas simplemente ligadas, doblemente ligadas, circulares
- Visualización interactiva de operaciones
- Laboratorio 1 (10%)

### Semanas 4-5: Pilas y Colas
- LIFO y FIFO con visualizaciones
- Operaciones básicas animadas
- Laboratorio 2 (15%)

### Semanas 6-7: Análisis de Algoritmos
- Notación Big O, Omega, Theta
- Comparación de complejidades
- Laboratorio 3 (10%)

### Semanas 8-10: Recursión
- Teorema Maestro
- MergeSort, QuickSort visualizados
- Laboratorio 4 (10%) + Parcial 1 (10%)

### Semanas 11-13: Árboles y Grafos
- BFS, DFS, Dijkstra
- Árboles BST, AVL, Heap
- Laboratorio 5 (15%)

### Semanas 14-15: Programación Dinámica
- Memorización vs Tabulación
- Problemas clásicos (Mochila, LCS, etc.)
- Parcial 2 (15%)

### Semana 16: NP Completitud
- Problemas P vs NP
- TSP, SAT, Ciclos Hamiltonianos
- Trabajo Final (10%) + Exposición (5%)

## 🚀 Uso

1. Abre `index.html` en tu navegador
2. Selecciona una unidad del menú lateral
3. Lee la teoría y explora los temas
4. Interactúa con las visualizaciones
5. Prueba los ejemplos en el laboratorio Python

## 💻 Tecnologías

- HTML5, CSS3, JavaScript
- Pyodide (Python 3.11 en WebAssembly)
- Visualizaciones con Canvas y DOM
