# 📚 Guía de Uso - Plataforma Lógica y Representación II

## 🎯 Estructura Mejorada

La plataforma ahora tiene **DOS versiones**:

### 1. `index.html` - Versión por Unidades
- Vista general de cada unidad completa
- Teoría consolidada de todos los temas
- Ejemplos generales
- Ideal para: Repaso rápido y visión general

### 2. `index-mejorado.html` - Versión por Temas Individuales ⭐ RECOMENDADO
- Cada tema tiene su propia página
- Teoría detallada específica
- Visualización interactiva dedicada
- Laboratorio con código completo
- Ideal para: Estudio profundo tema por tema

## 🚀 Cómo Usar la Versión Mejorada

### Navegación
1. **Menú lateral** con 7 unidades principales
2. **Click en una unidad** para expandir sus subtemas
3. **Click en un subtema** para ver contenido detallado

### Pestañas por Tema
Cada tema tiene 3 pestañas:

#### 📖 Teoría
- Definiciones completas
- Explicación de cada método
- Complejidades temporales y espaciales
- Ventajas y desventajas
- Casos de uso

#### 🎨 Visualización
- Animación interactiva del tema
- Controles para probar operaciones
- Explicaciones paso a paso
- Indicadores visuales de complejidad

#### 💻 Laboratorio
- Código Python completo y ejecutable
- Ejemplos prácticos
- Botón para abrir en el editor
- Ejecutar y ver resultados en tiempo real
